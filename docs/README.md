# Docs

This folder holds the canonical specification and type references. The **reference folder** (source of the spec) is read-only; this repo is the only place that gets edited.

- **CANONICAL_BUILD_PROMPT_TITAN_SENTINEL.txt** — Copied full spec for GRACE-X TITAN™ + SENTINEL™ (UI + behaviour). Use it as the single source of truth for what must be implemented.
- **VerificationReports.d.ts** — Type definitions for identity/verification (Stripe Identity VerificationReport). Used by the stub verification module; no external APIs are called (offline-first).
